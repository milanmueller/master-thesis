\section*{Erklärung zur KI Nuztung}

\bigskip
\bigskip
\bigskip

%
Hiermit erkläre ich, dass ich mit Genehmigung meines Prüfers die
generativen KI-Tool Claude Code des Herstellers Anthropic, mit den
Sprachmodellen Opus 4.5 und Sonnet 4.5, verwendet habe.
Claude Code wurde verwendet um Werkzeuge und Automatisierungen wie
Build Scripts und lokale Entwicklungsumbegungen, sowohl für das
Isabelle Project als auch für den LaTeX Quellcode der Arbeit, zu generieren.
Außerdem wurde Claude Code genutzt um potenzielle Randfälle und
Gegenbeispiele der Isabelle Formalisierungen zu finden.
Zudem wurde Claude Code nach Feedback zu bestehenden Formulierungen
der Arbeit gefragt, KI-generierte Sätze wurden jedoch nie direkt übernommen
sondern lediglich als alternative Formulierungen betrachtet um
potentielle semantische Unklarheiten zu erkennen. \\
Jeder Satz der Arbeit wurde vom Autor verfasst.\\
Des Weiteren bestätige ich ausdrücklich, dass diese Tools nicht
zur Generierung von Daten, als Quelle für sachliche Informationen
oder zur Erstellung von Inhalten dieser Arbeit verwendet wurden.
