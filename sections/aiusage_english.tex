\section*{Declaration of AI Usage}

\bigskip
\bigskip
\bigskip

I hereby declare that, with the approval of my examiner, I have
employed the generative AI tool Claude Code by Anthropic, using the
models Opus 4.5 and Sonnet 4.5 during the preparation of this thesis.
Claude Code was used to generate tooling (such as build scripts and
  development environments for both the Isabelle project as well as the
LaTeX code for the thesis), and was tasked to find edge cases and
counterexamples for the Isabelle implementation. In addition, Claude
Code was also asked for feedback on specific sentences of the thesis.
Proposed AI generated sentences were never adopted directly but
rather considered as alternative phrasings to detect semantically
ambiguous sentences.

I expressly confirm that AI tools were not used to generate data or as a
source of factual information or content for this thesis.
